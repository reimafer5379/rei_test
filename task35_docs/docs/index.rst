Краткое описание
----------------

Игра "Light 'em up!" представляет собой головоломку, где игрок должен соединить все
трубы на игровом поле, чтобы зажечь все блоки.

**Основные компоненты:**
- Стартовый экран (:class:`Startscreen`) - выбор размера поля
- Игровое поле (:class:`Game`) - основная игровая механика
- Система таймера - ограничение времени на решение
- Анимация правильного решения
